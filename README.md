# AroundMe

AroundMe is an ASP.NET Web Forms project for building location-aware and nearby-context features.

Initial structure:

- `Default.aspx` - starter home page.
- `Default.aspx.vb` - VB.NET code-behind.
- `App_Code/AroundMeInfo.vb` - shared starter class.
- `web.config` - ASP.NET configuration.

Next development can add pages for nearby places, location search, maps, favorites, user settings, and administrative setup.
